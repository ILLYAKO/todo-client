## Coding Exercise Full-Stack

### Front end

### Node.js is required.

#### 1. Clone git repository from https://github.com/ILLYAKO/todo-client.git

#### 2. Go to todo-client directory and run npm clean install

```Bash
cd todo-client
npm ci
```

#### 3. Start react server

```Bash
npm start
```
